# Análisis de Ventas de Videojuegos 🎮

Este proyecto realiza un análisis exploratorio de datos (EDA) sobre un dataset de videojuegos para identificar tendencias de ventas por plataforma, región y calificación ESRB.

## 🚀 Tecnologías utilizadas
* Python 3.13
* Pandas y Numpy (Procesamiento de datos)
* Matplotlib y Seaborn (Visualización)
* SciPy (Pruebas estadísticas)

## 📊 Características del análisis
* Limpieza de valores ausentes.
* Comparativa de ventas entre regiones (NA, EU, JP).
* Análisis de correlación entre puntuaciones de críticos/usuarios y ventas.
* Pruebas de hipótesis estadísticas.
